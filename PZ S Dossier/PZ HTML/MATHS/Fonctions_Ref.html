I - Fonction valeur absolue :


1 - Définition


Définition :

La fonction valeur absolue est définie sur R par :

f(x) = |x|


On a donc f (x) = x si x ≥ 0 et f(x) = - x si x < 0

On remarque que sa représentation graphique est symétrique par rapport à l’axe des ordonnées on a donc f (-x) = f(x)


- Sens de variation


La fonction valeur absolue est décroissante sur ] - ∞ ; 0] et croissante sur [ 0 ; +∞ [

II - Fonction polynôme du second degré :


1 - Définition


Définition :

Une fonction polynôme du second degré est définie sur R par une expression de la forme ax^2 + bx + c avec a ; b et c des réel et a différent de 0

Sa représentation graphique est une parabole :
• Lorsque a < 0 la parabole est orienté vers la haut :



• Lorsque a > 0 la parabole est orienté vers le bas :



Le point le plus haut ( a < 0) ou le point la plus bas (a > 0) est appelé sommet S de la parabole.


2 - Expressions de la fonction :


Une fonction polynôme du second degré admet 3 expression :

• Une forme développé de la forme :

ax2 + bx + c


• Une forme factorisée de la forme :

a (x – x1 ) (x – x2)


• Une forme canonique de la forme :

a (x -α)2 + β


Plusieurs de ces éléments peuvent être retrouvés graphiquement :



Le point d’intersection entre la parabole et l’axe des ordonnée (s’il y en a un) est c, ici c = 5 Les coordonnées du sommet de la parabole sont (α ; β )



Les deux points d’intersections entre la parabole et l’axe des abscisses ( ici A et B ) (si ils existent) sont x1 et x2, ici x1 = -1 et x2 = 3

3 - Etude de la fonction :


Sens de variation :

Pour déterminer le sens de variation on utilise la forme canonique,

• Si a < 0, f est croissante ] - ∞ ; α ] et décroissante sur [ α ; +∞ [
• Si a > 0, f est décroissante ] - ∞ ; α ] et croissante sur [ α ; +∞ [
Signe d’une fonction polynôme du second degré :

Pour déterminer le signe d’une fonction polynôme du second degré on utilise sa forme factorisé et on dresse un tableau de signe ou plus simple on calcule delta à partir de la forme développée :

Δ = b^2 – 4ac


• Si Δ = 0 on calcule :
x0 =
-b/2a


• Si Δ > 0 on calcule :
x1 =
- b – √Δ/2a


x2 =
- b + √Δ/2a




4 - Résolution d’équations :


On est en mesure de savoir α et β si nécessaire à partir de la formule développée :

α =
-b/2a
et β =
- Δ/4a2


Pour résoudre un équation de la forme ax2 + bx + c = 0 :

○ On calcule Δ = b2 – 4ac

• Si Δ < 0 alors il n’y a pas de solutions réelles à cette équation

• Si Δ = 0 on calcule l’unique solution: x0 =
-b/2a


• Si Δ > 0 on calcule les deux solutions x1 =
- b – √Δ/2a
et x2 =
- b + √Δ/2a


○ A partir de ces résultat on peut factoriser la fonction :

Si Δ < 0 alors on ne peut pas factoriser

Si Δ = 0 on calcule l’unique solution x0 =
-b/2a
et on écrit la fonction : f (x) = a (x + x0)^2

Si Δ > 0 on calcule les deux solutions x1 =
- b – √Δ/2a
et x2 =
- b + √Δ/2a
et on écrit la fonction f(x) = a (x-x1) (x-x2)




5 - Propriétés supplémentaires :


Si on a une fonction ax2 + bx + c qui admet deux solutions (racines) x1 et x2 alors
la somme x1 + x2 =
-b/a
et le produit x1 × x2 =
c/a


On a déjà établi que α =
-b/2a
et β =
- Δ/4a2
,
donc les coordonnées du sommet S de la parabole sont : (
-b/2a
;
- Δ/4a2
)
