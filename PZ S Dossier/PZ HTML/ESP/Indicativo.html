3 - Verbos con diptongación :

La diptonga se aplica a todas las personas excepto a nosotros y vosotros

2 casos :

El -e- se convierte en -ie- como en el verbo pensar : pienso, piensas, piensa, pensamos, pensáis, piensan Otros ejemplos : querer : quiero ; preferir : prefiero ; entender : entiendo ; sentir : siento

El -o- se convierte en -ue- como en el verbo mostrar : muestro ; muestras ; muestra ; mostramos ; mostráis ; muestran


Otros ejemplos :
poder : puedo
dormir : duermo


4 - Verbos con debilitamiento (affaiblissement):

El debilitamiento convierte el -e- en -i- para todas las personas excepto nosotros y vosotros

Pedir : pido, pides , pide , pedimos , pedís , piden

Otros ejemplos :
repetir : repito
seguir : sigo
vestirse : me visto
