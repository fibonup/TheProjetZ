I - El Indicativo


1 - El Presente :

Existe una irregularidad únicamente a la primera persona :


• Caer : caigo, caes…
• Hacer : hago , haces…
• Poner : pongo, pones…
• Salir : salgo, sales…
• Traer: Traigo, traes…
• Decir : digo, dices…
• Oír : oigo, oyes… (atención para oír : él y) …oye , oímos, oís , oyen
• Tener : tengo, tienes …
• Venir : vengo, vienes…
• Dar : doy, das…
• Ver : veo, ves…
• Estar : estoy , estás, está, estamos, estáis, están
• Conocer : conozco, conoces…
• Saber : sé , sabes…
• Coger : cojo, coges…
• Construir : construyo , construyes… (misma cosa que oír)
